== Caractéristiques ==

== Potions ==
- Potion Tumvoiplu : Invisible pour un temps donnée ou jusqu'au prochain mouvement.
- Potion Oracle : Permet de voir les invisibles pour un temps donnée.
- Potion Kyexploz : Place une potion qui explose quand on la ramasse.
